# VISUAL_REFERENCE_LOCK — STRATEGICLAND

> The only acceptable references for visual, motion, atmospheric, and interactive decisions.
> If a reference is not on the **allowed** list, **do not use it**.
> If a reference is on the **forbidden** list, **flag and reject** any work derived from it.

---

## 1. Why This Lock Exists

Most "premium" web design fails because its references are **other premium web design**. The result is a closed loop: the same gradients, the same magnetic CTAs, the same hero-with-3D-blob, the same "Awwwards 2024 winner" recipe.

STRATEGICLAND's references are **outside the web entirely** — film, print, architecture, photography. This is not affectation. It is the operational protection against the collapse of the project's identity into the genre's average.

When in doubt, look at a 1965 Carlo Scarpa drawing, not at a 2024 portfolio site.

## 2. Allowed References (Whitelist — Authoritative)

### 2.1 Architectural studio practice
- **Simon Holm Studio** — restraint, weight of negative space, editorial type as architecture, hairline detail.
- **Vitra Design Museum** — exhibition typography, monograph layouts, asymmetric editorial composition.
- **Carlo Scarpa** — joinery as ornament; the way a hinge becomes the point of the room.
- **Kaare Klint** — proportional discipline; surface restraint; furniture-grade hierarchy.
- **Donald Judd** (writings on space, not just sculpture) — modular discipline, scale awareness.
- **Peter Zumthor** (*Atmospheres* book in particular) — material presence, sensory restraint.

### 2.2 Cinematic language
- **A24 title sequences** — pacing of silence, asymmetric composition, contemplative cuts.
- **Wong Kar Wai** (*In the Mood for Love*, *2046*) — restraint in motion, the held shot, color as emotion.
- **Roger Deakins** (*Sicario*, *Blade Runner 2049*, *1917*) — vignette weight, light direction discipline, atmospheric crush.
- **Christopher Doyle** — handheld imperfection that reads as human breath.
- **Terrence Malick** (*The Tree of Life*) — non-linear pacing, contemplative dwell time.
- **Wim Wenders** (*Wings of Desire*, *Paris, Texas*) — quiet observation, time as a material.
- **Lynne Ramsay** (*We Need to Talk About Kevin*) — composition under restraint.
- **Paul Thomas Anderson** (*The Master*, *There Will Be Blood*) — formal symmetry deployed as cinematic weight.
- **Documentary cinema** broadly — *Leviathan* (Castaing-Taylor), *Manakamana*, *Sweetgrass* — the long, fixed, observational frame.

### 2.3 Editorial print
- **Apartamento** — copy that breathes off-baseline; images that sit slightly off-grid; intimacy as a layout principle.
- **Aperture** — silver-gelatin reverence; image-as-artifact; caption discipline.
- **MacGuffin** — single-subject obsession; editorial restraint; warm paper stocks.
- **The Gentlewoman** — interview-driven layouts; sans-serif clarity within editorial composition.
- **032c** — punk-editorial architecture; aggressive restraint; serif-as-instrument.
- **Toilet Paper Magazine** (Maurizio Cattelan / Pierpaolo Ferrari) — image-as-statement; the deliberate frame.
- **Migrant Journal** — atmospheric editorial design; field photography reverence.
- **Real Review** — architectural editorial; opinion typeset as weight.

### 2.4 Atmospheric photography
- **Hiroshi Sugimoto** — *Seascapes*, *Theaters*, *Conceptual Forms*. Slow drift, near-stillness, infinite depth.
- **Sebastião Salgado** — silver gelatin grain, depth, warmth, dignity in imperfection.
- **Saul Leiter** — color restraint, painterly weight, off-center composition.
- **Rinko Kawauchi** — soft light, ambient palette, intimate distance.
- **Andreas Gursky** — large format scale, atmospheric layering.
- **Wolfgang Tillmans** — print-and-pin layouts, intentional informality at scale.

### 2.5 Atmospheric editorial minimalism (web — narrowly)
- **Studio websites that pre-date the Awwwards-aesthetic homogenization** — circa 2008–2014 boutique studio sites.
- **Specific practitioners (sparingly):**
  - **Bureau Cool** — restraint as policy.
  - **Studio Dumbar** archives — type-as-architecture web work.
  - **Foreign Policy Design** archives — editorial-system web work.
- **Note:** these are the **only acceptable web references**, and only when used as proof that web can hold print-grade restraint.

## 3. Forbidden References (Blacklist — Authoritative)

If a design candidate is "inspired by" any of these, **reject the candidate**.

### 3.1 Startup / SaaS premium aesthetics
- ❌ Vercel, Linear, Stripe, Notion marketing sites — clean, optimized, friendly. We are **not** friendly. We are **deliberate**.
- ❌ Apple product launch pages — modular hero + scrolling reveal + "Pro" type. We are **not** a product launch.
- ❌ "Tech-forward" portfolios on Awwwards 2020+ — magnetic CTAs, 3D blobs, gradient meshes, scroll-triggered transforms.
- ❌ Webflow / Framer template aesthetic — every "premium" template uses the same hero + 3-column features + testimonial-grid + CTA-strip pattern.

### 3.2 Generic Awwwards SOTD aesthetic
- ❌ "Brutalist" portfolio sites (~2018–2020) with Helvetica + magenta + grid distortion.
- ❌ "Cinematic agency" sites with full-bleed hero video + bold sans + slow zoom.
- ❌ "Studio" sites that clone the Locomotive Mtl / Active Theory / Resn template with Lenis + GSAP + "click anywhere to enter".
- ❌ Anything that places its motion library credit ("Built with GSAP / Lenis / Three.js") in the footer as a brag.

### 3.3 Cyberpunk / crypto / hyper-tech AI
- ❌ Neon green + magenta on black with monospace.
- ❌ "AI-native" branding — gradient logos, particle backgrounds, "intelligence" as a CTA.
- ❌ Crypto / Web3 aesthetic — chromatic gradients, glitchy type, "join the mission" hero.
- ❌ Cyberpunk dystopia visuals — Blade Runner 2077 / Tron / Neuromancer mood boards.
- ❌ "Quantum" / "neural" / "matrix" visual metaphors of any kind.

### 3.4 Trend-driven decoration
- ❌ Glassmorphism — frosted card UI. **Strictly forbidden.** (See `atmosphere-system.mdc`.)
- ❌ Neumorphism — soft 3D extrusion shadows. **Strictly forbidden.**
- ❌ "Liquid" or "fluid" page transitions with WebGL-on-image distortion.
- ❌ "Stretch text" / "elastic text" / "physics text" hero animations.
- ❌ Cursor trails, particle followers, sparkle effects.
- ❌ "Auto-scroll narrative" sites (the user can't stop scrolling).
- ❌ Gradient mesh backgrounds (Stripe-style multi-color grad).
- ❌ Lottie animations of any kind in editorial sections.
- ❌ Dark-mode toggles. The site has **one** mode (atmospheric). No toggle.

### 3.5 Motion overload
- ❌ Every element animates on entry.
- ❌ Hover states scale 1.05+ with shadow elevation.
- ❌ "Reveal-on-scroll" applied indiscriminately to every element.
- ❌ Marquee tickers as decoration.
- ❌ Auto-rotating image carousels.
- ❌ Click-to-flip cards with reveal animations.
- ❌ Pulsing CTAs.
- ❌ "Fun" micro-interactions on form inputs.

### 3.6 Generic luxury minimalism
- ❌ Centered-everything composition.
- ❌ Pure black + pure white + one accent color (the "agency luxury" cliché).
- ❌ Helvetica Neue + huge image hero + thin caption (the "Apple-but-not-Apple" cliché).
- ❌ "Refined", "sophisticated", "elevated" as design adjectives — these are marketing language, not design language.
- ❌ Faux-Swiss grid sites that don't earn the rigor.

### 3.7 AI-generated aesthetic
- ❌ Midjourney "ethereal portrait" hero imagery — recognizable plastic skin, generic dreaminess.
- ❌ Stable Diffusion "abstract concept art" backgrounds.
- ❌ DALL-E pastel illustrations in editorial sections.
- ❌ Any imagery that the audience can pattern-match to "AI-generated".

### 3.8 Specific anti-references (sites / studios to never imitate)
- ❌ Any site that won Awwwards Site of the Day in the last 18 months whose aesthetic is "minimal cinematic with magnetic CTAs and 3D shader blob".
- ❌ Any "Web3 / Crypto / DeFi" site, including the well-designed ones.
- ❌ Any "Generative AI startup" site (the entire genre).
- ❌ Any site whose primary hook is its tech stack (sites that brag about Three.js usage).

## 5. Reference Application Rules

### 5.1 How to use references
- **Study the principle, not the surface.** "Sugimoto's seascapes" means *long observation, near-stillness, atmospheric depth* — not *literally a sea image*.
- **Cite references in design reviews.** "This decision references Roger Deakins' vignette weight in *Sicario*" is a valid argument. "It looks cool" is not.
- **One reference per decision.** If a candidate cites three references, two are decoration. Pick the strongest one.
- **References do not stack.** A "Sugimoto + A24 + Apartamento" hero is not "more cinematic". It is incoherent. Pick one.

### 5.2 How NOT to use references
- ❌ Referencing a film without naming the cinematographer is shallow — *which* shot, *which* sequence, *which* lighting decision?
- ❌ Referencing "the Apartamento aesthetic" without specifying *which issue, which spread, which layout principle*.
- ❌ Citing a moodboard as a reference. Moodboards are inputs, not outputs.

### 5.3 Reference fatigue
A reference loses power if used too often. **Rotate references** across chapters:
- HC-04 (Cultural) leans on Sugimoto + Sebastião Salgado.
- HC-05 (Human) leans on Wim Wenders + Saul Leiter + Apartamento.
- HC-06 (Case study) leans on documentary cinema + 032c.
- HC-01 / HC-02 / HC-03 (Hero variants) lean on A24 + Roger Deakins.

Repeating the same reference in 4 consecutive chapters means the reference has become the **default**, which is the exact failure mode this lock is preventing.

## 6. The Reference Test (Apply Before Committing Design Work)

For any visual / motion / atmospheric / interaction design, answer in writing:

1. **What's the reference?** (Specific work, specific scene, specific spread.)
2. **Which principle from that reference is being applied?** (One sentence.)
3. **Is the reference on the allowed list (§2)?** Yes / no.
4. **Is the reference NOT on the forbidden list (§3)?** Yes / no.
5. **Could you defend this reference in a creative-direction review?** Yes / no.

Five "yes"es → ship it.
Any "no" → revise.

---

## 7. Reference Drift Detection

Indicators that the project has drifted into forbidden reference territory:

- A stakeholder describes the site as "premium" or "luxury" without prompt.
- A new contributor's first instinct is to add a magnetic CTA.
- The hero "needs a glassmorphic card" for legibility (it doesn't — fix the atmosphere).
- A chapter is described as "engaging" or "fun".
- A motion is justified as "modern" or "contemporary".
- The team starts looking at Awwwards SOTD for inspiration.
- A review uses words like "sleek", "polished", "vibrant", "dynamic", "innovative".

If two or more of these indicators appear in a single sprint, **schedule a reference recalibration** — re-read this file aloud as a team, and re-screen the canon HTML files alongside Sugimoto / Wong Kar Wai stills.

---

## 8. Forbidden Reference Recovery

If forbidden-reference work has already shipped:

1. **Identify** — flag the offending element with a `// REF-LOCK-VIOLATION` comment in the code.
2. **Quarantine** — disable the element in production (feature flag) without removing the code yet.
3. **Re-design from allowed references** — start from the allowed list, not from "fixing" the forbidden one.
4. **Replace and remove** — once the new version ships, delete the violation code.
5. **Document the lesson** — add a note in `MASTER_STATE.md §3 Decisions logged` explaining what failed and why.

## 10. Reference Lock Maintenance

This file is reviewed every 6 months. Updates may:

- **Add to the allowed list** if a new reference passes the DNA Test (`DESIGN_DNA.md §5`).
- **Add to the forbidden list** as new aesthetic trends emerge that conflict with the project DNA.
- **Never remove from the forbidden list** without creative-direction sign-off and a recorded rationale in `MASTER_STATE.md`.

Any contributor who notices a drift trend (per §7) may propose a forbidden-list addition via PR.

---

## 11. The One-Sentence Filter

If you can't decide whether a reference is appropriate, ask:

> *Would Hiroshi Sugimoto, Wim Wenders, Carlo Scarpa, and the editor of Apartamento all nod approvingly if they saw this on the page?*

If even one of them would frown, the reference is wrong.

# wmn/nd — We Make Noise / Not Digital

Sitio de la consultoría wmn/nd. Página única (single-file HTML) con Three.js (túneles WebGL) y GSAP (animaciones de scroll).

## Estructura
- `index.html` — sitio completo (HTML + CSS + JS inline)
- `assets/` — imágenes de los casos de IMPACTO (ver LEEME.txt)

## Despliegue
Sitio estático. Compatible con Vercel, Netlify o GitHub Pages sin build step.
